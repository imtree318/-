# 湖北省利川市建南王氏家族简介

```
   湖北省利川市建南王氏家族，是广东乳源县元发公后人，于清朝初年，随着“湖广填四川”移民大潮，先迁居于四川省顺庆府邻水县，再迁至湖北利川建南安家落户，艰苦创业，繁衍生息，逐渐发展成本地名门望族。先后修建了王家祠堂，重新制定了建南王氏40字字派，并续修了族谱。今天为止，已经发展到15字辈，家族人数也已数千人。
   湖北利川建南地处湖北西南角、属于偏远山区，由于经济比较落后，交通不便，信息蔽塞，建南王氏后人逐渐断绝了与四川邻水家人、广东乳源家人的联系，期间也曾修过族谱，但也是一百多年前了。今天，我们成立了族谱编委会，旨在重修族谱、追根求源、传承家风、 凝聚族人，并望重新建立起与邻水、乳源宗亲的紧密联系。目前，编谱相关工作已展开，预计在今年底编印完成。
   由于可参资料有限，加上间隔太过久远，我们至今尚未完全弄清从乳源到邻水、从邻水到利川建南的全部传承线索，希望乳源和邻水宗亲提供相关资料和信息，帮助我们完善，不胜感激。
   我们的40字派是从王崇仁之子王福忠开始的。王福忠字义臣，首字为“义”字，40个字辈如下：义礼建家先、明允会贞元、天道齐余福、化云翠益玄、有恒可作圣、唯敏诗能贤、至诚常悠久、一忠万世传。
```
